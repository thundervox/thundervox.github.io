screenOffset()

現在の論理スクリーン関するディスプレイのビューポートを移動します。screenOffset はプラットフォームゲームの背景スクロール処理で利便性があります。従来の offset() コマンドも引き続き使用可能です。 

文法:
screenOffset ( x , y )
offset ( x , y )

用例:
screenopen(0,1280,480,0,0,640,480,0)
for i = 0 to 100
    ink(rgb(rnd(255),rnd(255),rnd(255)))
    line(rnd(1280),rnd(480),rnd(1280),rnd(480))
next
x=0
while x<640
    x+=2
    screenOffset( x , 0 )
    waitvbl
end while


用例:
screenOffset.sdlbas

screenOffsetX および screenOffsetY コマンドを参照してください。
